projects

Document Sharing System - november 2018

Python, Tkinter

Implemented a document sharing system, where multiple users can work on the same document. Three users are superuser, ordinary user, and guest user, the super user is the administrator whereas all other users who can create a new document are called ordinary users and guest users are only able to view documents. Users can also write complains which are managed by super user and any user can be promoted to a super user after being approved by another super user. A list of the taboo word is maintained where all false words are listed and these words cannot be used inside the document. A class hierarchy is maintained as all super user are ordinary users and all ordinary users are a guest user, view project
